# acync_opt_kz_parce
Парсинг сайта otp.kz ассинхронно на NodeJS с помощью cheerio, axios, xlsx

## Библиотеки

```sh
axios
cheerio
xlsx
fs
```
## Инструкция 

```sh
cd Desktop
git clone https://github.com/Zhangirkhan/acync_opt_kz_parce.git
cd acync_opt_kz_parce
npm install
node index.js
```
## Формат экспорта

XLSX (excel)
